# School Images

## Logo files

| File | Use |
|------|-----|
| `public/images/logo.png` | Website — navbar, footer, on-page display |
| `public/logo.svg` | Browser tab, favicon, SEO icon metadata |
| `public/favicon.svg` | Compact shield emblem (small tab sizes) |

Replace `logo.png` with a higher-resolution export if needed.  
Replace `logo.svg` with your official vector file for pixel-perfect favicons.

## Other photos

- `director.jpg` — Director photo
- `manager.jpg` — Manager photo
- `faculty/` — Individual teacher photos
- `toppers/` — Topper / scholar photos
- `school-building.jpg` — Hero / campus photo

Update `image` fields in `data/faculty.ts` and `data/toppers.ts` when adding real photos.
